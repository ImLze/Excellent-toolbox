### deployment server
在你的服务器上部署``Excellent toolbox``<br>
- 下载此仓库的[``dev``](https://github.com/ImLze/Excellent-toolbox/archive/refs/heads/dev.zip)分支 (直接点击此链接下载)<br>
- pnpm install<br>
- pnpm start<br>


### Add Tools
格式:
```
tools_name: # //工具名字
tools_links: # //工具链接
tools_support: # //工具语言
tools_readme: # //工具介绍
```
